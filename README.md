# YouTube-Unhook-Auto
I wanted to automatically manage the open and closing of a browser extension called 'Unhook' with a python script. It is used to help you stop wasting time and getting distracted so you can focus on real work. 

NOTE: I did not make this browser extension. I am just adding external functionality with a python script.

Unhook: https://addons.mozilla.org/en-US/firefox/addon/youtube-recommended-videos/ 

I do not recommend to use this script all the time as I didnt manage the memory resources well so it can be very slow haha.
